Ribbons.Guide is a progressive web app that allows you to keep track of your Pokémon and their Ribbons! Ribbons.Guide doesn't store or send any data anywhere—it's all saved on your device! This is an open-source project hosted on GitHub using GitHub Pages. It was developed as a love letter to the helpful and welcoming r/PokémonRibbons community.

The images used in Ribbons.Guide were provided by PokéSprite, ポケモン徹底攻略, PokéLiberty, Pokémon GO Assets, and Material Symbols.
Ribbons.Guide also makes use of jQuery, jQuery UI, jQuery UI Touch Punch, Selectize, UpUp, and Google Fonts.
